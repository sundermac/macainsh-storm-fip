# Couples Resort - FIP 🏝️💖
The Level 1 Interactive Media Design FIP is a redesign of the Couples Resort website. It builds upon the in-class build, adding three mobile and one desktop pages.

## 🎯 Project Objectives
- Build upon the class version of the Couple's Resort website.
- Complete **three additional mobile pages** based on the main navigation.
- Create **one desktop page** with original content that is consistent with the existing site.
- Use the **static CSS Grid** for both mobile and desktop layouts.
- Use the **::after pseudo-selector** to add a graphic element on one page.
- Create a simple **CSS animation** for a submit button on one page.
- Implement the **show/hide functionality** using a hamburger menu or the details and summary elements.

## 💾 Installation
No installation required. Download the zip file and extract it.

## 🌐 Usage
Open the `index.html` file in the browser of your choice.
